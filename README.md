## ğŸ Quick Start

### 1. Prerequisites
- Node.js (v18+)
- MongoDB connection string
- Vercel CLI (for deployment)

### 2. Installation
Clone the repository and install dependencies in both folders:
```bash
# Setup Backend
cd Backend/Parcel_Delivery_System
npm install

# Setup Frontend
cd ../../Frontend
npm install
```

### 3. Running Locally
Refer to the individual READMEs in the subdirectories for detailed environment variable setup.
- **Backend**: `npm run dev` (starts on port 5000)
- **Frontend**: `npm run dev` (starts on port 5173)
